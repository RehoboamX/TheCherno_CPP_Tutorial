# Lesson8 变量
#### 变量允许我们命名我们储存在内存中的数据，使得我们可以继续使用它。
(1)当我们创建变量时，该变量会被储存在**内存**(**memory**),它可能会被储存在两个地方：**栈**(**stack**)或**堆**(**heap**)。  
(2)在c++中，有很多基本数据类型，这些基本数据类型基本上包含了我们在程序中储存的任何种类的数据。不同变量类型本质上都是二进制的体现，所以其之间的唯一区别是大小，即这个变量会占多少内存。   
(3)对整型(integer)来说，其关键字为**int**，整型是常规意义上4个字节的数据类型。数据类型的真实大小实际上取决于编译器，而不同的编译器可能会有不同。int为有符号整型，而无符号整型为**unsigned int**。  
(4)简要介绍几种基本的数据类型，**char**储存1个字节的数据，**short**储存2个字节的数据，**int**储存4个字节的数据，**long**通常也储存4个字节数据（但具体要取决于编译器），**long long**通常保存8个字节数据。也有一些数据类型像是**long int**，或是将**unsigned**放在之前的任意一个前面，有很多不同的组合，但我们要着重介绍的，是前面5种基本的数据类型。  
- **char**通常拥有储存字符，不仅仅是数字（也可以给char赋值数字）。我们把‘A’或是整数65(A的ASCII码)赋值给char变量时，打印结果都是A。这是因为在把char变量传给cout，其会把这个char变量看成是一个字符而非数字。  
- 类似的如果我们把‘A’赋值给**short**或**int**等数据类型，打印时会打出65，因为cout不把这些形式的变量看作字符而是看作数字。  
- 如**float**，**double**（或**long double**等）用来储存小数。**float**类型的数据占4个字节,**double**类型数据占8个字节。※需要注意的是：当我们定义float a = 5.5；这里面的5.5实际上是一个double类型的变量，然后在把这个double类型的值赋给float类型，这样就会出现精度丢失。如果要直接指定生成float类型的数字，要在数字后加上**F**或**f**。如：float a = 5.5f；
- **bool**代表布尔类型的变量，占1个字节。其值有两种：true或是false。如果给bool类型的变量赋值false或0，打印时会得到0；如果赋值true或任何非0的数，打印会得到1。尽管bool类型储存0或1只需要一位就够了，但是为了能够寻址，我们只能用一个字节的大小去储存它。而且我们可以通过一些技巧，在一个字节上存储8个bool变量。  

(5)数据类型的使用取决于编程者，有一些既定的约定，但又没有什么具体的规则你必须遵循。c++中没有过多的规则限制你，因此这些数据类型的唯一区别就是定义这些变量时分配的内存大小。  
(6)数据类型占的大小与不同的编译器有关，我们可以通过调用**sizeof**操作符来查询变量占用内存的大小。sizeof(变量类型/名称)会返回占用的字节数。  
(7)在这些基本类型的基础上，我们还可以将他们转换为**指针**(**pointers**)或**引用**(**references**)。  
- **指针**(**pointer**)的用法是在变量类型后加一个*，如int* a = 2;
- **引用**(**reference**)的用法是在变量类型后加一个&，如int& a = 2;
